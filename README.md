# Student Management System

A simple menu-driven Student Management System built using Python.

## 🚀 Features
- Add Student
- View Students
- Update Student
- Delete Student
- Data stored using file handling

## 🛠️ Technologies Used
- Python
- File Handling

## ▶️ How to Run
1. Clone the repository
2. Open terminal
3. Run:
   python student.py

## 📌 Project Description
This project allows users to manage student records using a menu-based interface. It performs CRUD operations and stores data in a text file.
